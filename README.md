# ASIX - OPT - Programming of Things

## Mòdul Optatiu - Python - Programació de les Coses - ASIX

Apunts per a donar el mòdul Optatiu de Programació de les Coses al Cicle Administració de Sistemes Informàtics en Xarxa - ASIX.

*Aquests apunts estan fets en entorn `mkdocs` i plantilla `material`.* v1.

El que es pretén és mostrar a l'alumnat la potència i facilitat d'executar processos per a realitzar tasques pròpies de l'administració de sistemes en llenguatge python.

Per això es pretén arribar a entendre 

* l'entrada i tractament d'informació numèrica i strings,

* les diferents opcions per emmagatzemar molta informació en memòria:

    * llistes

    * diccionairs

    * tuples i conjunts

* la lectura i escriptura d'informació en i des de fonts externes:

    * fitxers

    * bases de dades

    * URL

* l'ús de llibreries d'altres per facilitar les tasques a administrador d'ASIX

    * os, shutil, subprocess
    
    * argparse, psutil, pathlib

    * requests, paramiko

    * logging

### Parts pels apunts

* Tractament de la informació:

    * Variables: Entrada i sortida d'informació

    * Tipus de dades: números i strings
