# POT - ESP-32- La funció Timer

La funció `Timer` de la llibreria `machine` permet a l'ESP32 executar una funció específica (anomenada *callback*) a intervals de temps exactes i predefinits, sense aturar el bucle principal del programa.

Quan utilitzem `#!python time.sleep()`, l'ESP32 es queda bloquejat (no pot fer res més) durant aquest temps. Amb `Timer`, l'ESP32 executa la funció de temporitzador quan toca, però continua executant el codi principal.

## Paràmetres Clau del Timer

Quan inicialitzem un temporitzador, normalment establim:

* `period` (**Període**): L'interval de temps entre execucions del callback, especificat en mil·lisegons (ms).

* `mode` (**Mode**):

    * `Timer.ONE_SHOT`: La funció s'executa només una vegada.

    * `Timer.PERIODIC`: La funció s'executa repetidament cada període. (Aquest és el que necessitem).

* `callback` (**Funció a executar**): La funció de Python que s'ha d'executar quan el temporitzador caduca.

```python title="LEDs Alternants amb Timer"
from machine import Pin, Timer

# --- 1. Configuració ---
PIN_VERMELL = 18
PIN_VERD = 19
TEMPS_MS = 1000  # 1000 mil·lisegons = 1 segon

# Inicialització dels Pins (Sortida)
led_vermell = Pin(PIN_VERMELL, Pin.OUT, value=0)
led_verd = Pin(PIN_VERD, Pin.OUT, value=1) # Comencem amb el Verd encès

# Variable per fer el seguiment de quin LED està actualment encès

# --- 2. Funció Callback del Temporitzador ---
def alternar_leds(timer):
    """
    Aquesta funció s'executa automàticament cada TEMPS_MS (1000ms).
    """
    
    led_verd.value((led_verd.value() + 1) % 2)
    led_vermell.value((led_vermell.value() + 1) % 2)

# --- 3. Inicialització i Arranque del Temporitzador ---
# Crea un objecte Timer. A l'ESP32, pots fer servir Timer(-1) per utilitzar el primer temporitzador disponible.
timer = Timer(-1)

# Inicialitza el temporitzador
timer.init(
    period=TEMPS_MS,          # Interval de 1000ms (1 segon)
    mode=Timer.PERIODIC,      # Repetició contínua
    callback=alternar_leds    # La funció a executar
)

print("Temporitzador iniciat. El programa principal pot continuar executant altres tasques.")

# --- 4. Bucle Principal (No Bloquejant) ---
# Ara, el bucle principal pot fer altres coses lliurement:
i = 0

    # Aquest missatge es printarà RÀPIDAMENT, mentre el LED només canvia cada segon
    # Això demostra que el programa no està BLOQUEJAT per l'espera del LED.
    # time.sleep(0.1)  # (Opcional) Un petit sleep per no omplir la consola
```
