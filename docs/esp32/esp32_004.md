# POT - ESP-32- Les interrupcions Externes

Una interrupció és l'aturada, temporal, de l'execució normal d'un procés per passar a executar una subrutina. Quan aquesta subrutina finalitza, es continua amb el programa on s'havia deixat.

Connectarem 3 leds, un de cada color i els anirem encenent d'un en un. A més, posarem un pulsador que quan el premem, voldrem que encengui, durant un segon, tots tres llums. Si ho programem de forma lineal, només podrem comprovar l'estat del pulsador en alguna línia de codi, o en moltes...

=== "Una comprovació"

    ```py
    from machine import Pin
    from time import sleep

    bt = Pin(19, Pin.IN, Pin.PULL_DOWN)
    ledR = Pin(25, Pin.OUT)
    ledG = Pin(26, Pin.OUT)
    ledB = Pin(27, Pin.OUT)
    ledR.off()
    ledG.off()
    ledB.off()
    i = 0
    while True:
        ledR.on()
        ledG.off()
        ledB.off()
        sleep(0.5)
        ledR.off()
        ledG.on()
        ledB.off()
        sleep(0.5)
        ledR.off()
        ledG.off()
        ledB.on()
        sleep(0.5)
        ## comprovació del pulsador
    ```

En el primer cas, només es detectarà el botó després de cada seqüencia de Red - Green - Blue, és a dir just quan s'apagui el blau i abans d'encendre's el verd.

En el segon cas, fem massa comprovacions i potser ens n'hem deixat alguna, el que està clar és que, mentre tenim un llum encès, no es fa cap comprovació.

Problemes

* Falta immediatesa.

* potser no detectem la pulsació, ja que en aquell moment no estem executant la instrucció necessària

La placa ESP-32 ens permet tractar:

* interrupcions externes: es produeixen en resposta a un event extern: pulsador (canvi estat en un GPIO), sensor, etc.

* interrupcions internes: es produeixen en un event programat: temporitzador, excepció del programa, etc.

Ens centrarem en les interrupcions externes que poden utilitzar-se en qualsevol Pin GPIO de la place ESP32.

## machine.Pin.irq

```py
 Pin.irq(handler=None, trigger=Pin.IRQ_FALLING | Pin.IRQ_RISING, *, priority=1, wake=None, hard=False)
```
 
 La funció [`machine.Pin.irq`][irq] configura un *gestor d'interrupcions* que es cridarà quan la font de disparador del pin estigui activa. Si el mode pin és `Pin.IN`, la font de disparador és el valor extern del pin. Si el mode pin és `Pin.OUT` la font de disparador és el buffer de sortida del pin. En cas contrari, si el mode pin és `Pin.OPEN_DRAIN`, la font de disparador és el buffer de sortida per a l'estat '0' i el valor extern del pin per a l'estat '1'.

Crearem una funció `encenLlums` que encendrà tots els pins durant 5 segons i després deixarà tot com estava.

Executarem el següent codi però veurem que només s'encenen els tres llums si tenim apretat el botó després que s'apagui el blau. Entre la resta de colors, no passa res.

```py hl_lines="4-14 38 39"
from machine import Pin
from time import sleep

def encenLlums():
    vledR = ledR.value()
    vledG = ledG.value()
    vledB = ledB.value()
    ledR.on()
    ledG.on()
    ledB.on()
    sleep(1)
    ledR.value(vledR)
    ledG.value(vledG)
    ledB.value(vledB)

bt = Pin(19, Pin.IN, Pin.PULL_DOWN)
ledR = Pin(25, Pin.OUT)
ledG = Pin(26, Pin.OUT)
ledB = Pin(27, Pin.OUT)
ledR.off()
ledG.off()
ledB.off()
i = 0
while True:
    ledR.on()
    ledG.off()
    ledB.off()
    sleep(0.5)
    ledR.off()
    ledG.on()
    ledB.off()
    sleep(0.5)
    ledR.off()
    ledG.off()
    ledB.on()
    sleep(0.5)
    ## comprovació del pulsador
    if bt.value() == 1:
        encenLlums()
```

Ara, doncs, afegirem la interrupció per tal que s'executi aquest codi quan s'apreti el botó (Pin 19), independentment del moment on estiguem. Per això afegirem la línia:

```py
bt.irq(handler=encenLlums, trigger=Pin.IRQ_RISING)
```

que defineix una interrupció: quan el *botó (`bt`)* passi d'estat baix a estat alt *`RISING`*, executarà la funció `encenLlums`, quedant el codi de la següent manera:

```py hl_lines="4-14 17"
from machine import Pin
from time import sleep

def encenLlums(_):
    vledR = ledR.value()
    vledG = ledG.value()
    vledB = ledB.value()
    ledR.on()
    ledG.on()
    ledB.on()
    sleep(1)
    ledR.value(vledR)
    ledG.value(vledG)
    ledB.value(vledB)

bt = Pin(19, Pin.IN, Pin.PULL_DOWN)
bt.irq(handler=encenLlums, trigger=Pin.IRQ_RISING)
ledR = Pin(25, Pin.OUT)
ledG = Pin(26, Pin.OUT)
ledB = Pin(27, Pin.OUT)
ledR.off()
ledG.off()
ledB.off()
i = 0
while True:
    ledR.on()
    ledG.off()
    ledB.off()
    sleep(0.5)
    ledR.off()
    ledG.on()
    ledB.off()
    sleep(0.5)
    ledR.off()
    ledG.off()
    ledB.on()
    sleep(0.5)
```

!!!tip "Prova de variar el temps que tenim encesos els Leds per veure bé el funcionament de la interrució"

Si premem diverses vegades seguides el botó, s'aniran acumulant totes les crides a la interrupció i per tant veurem que dura força més els llums encesos.

Per això, podem, quan ens interessi, activar o desactivar la interrupció quan ens calgui.

```py
Pin.irq(handler=None)
```

Al nostre exemple seria així

```py hl_lines="2 13"
def encenLlums(_):
    bt.irq(handler=None)
    vledR = ledR.value()
    vledG = ledG.value()
    vledB = ledB.value()
    ledR.on()
    ledG.on()
    ledB.on()
    sleep(5)
    ledR.value(vledR)
    ledG.value(vledG)
    ledB.value(vledB)
    bt.irq(handler=encenLlums, trigger=Pin.IRQ_RISING)
```

!!!note "Fixeu-vos que fins la funció cal rebre almenys un paràmetre. Nosaltres hem posat `_` que indica que no el necessitem però sempre rebrem l'objecte sobre el que s'executi al interrupció"

    en el nostre cas, rebriem l'objecte del botó `bt`.
