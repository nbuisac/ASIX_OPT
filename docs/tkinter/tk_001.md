# [Canvas][]

[*Canvas*][Canvas] és el giny de [Tkinter][] que podem utilitzar per dibuixar-hi a sobre. Ens serveix com un full al qual hi podem dibuixar diferents elements com ara: línies, cercles, quadrats, etc.

Algunes de les propietats que té són:

* `height` pixels

* `width` pixels

* `background` color

* `foreground` color

Cal posar el Canvas dins una finestra, com a mínim.

```py linenums="1"
from tkinter import *

finestra = Tk()

c = Canvas(finestra)
```

A parir d'ara podem pintar-hi 

Podem utilitzar la comanda `dir(<objecte>)` per veure'n les possibles propietats i mètodes.

```py
dir(c)
```

## [`create_line`][create_line]

Per crear una o vàries línies seguides.

```py
c.create_line(100, 300, 400, 300, 250, 100, 100, 300, arrow=LAST, dash=(5, 1, 2, 1), dashoff=3, fill="blue", width=5)
```

Hem creat un triangle, indicant els punts pels que hem de traçar la línia. Fixa't que al dibuixar línies, hem hagut d'indicar el punt inicial al principi i al final.

## [`create_rectangle`][create_rectangle]

Per crear un rectangle donant els dos vèrtex.

```py
c.create_rectangle(100, 300, 400, 100, fill="#00ffff", outline="red")
```

## [`create_polygon`][create_polygon]

Per crear un polígon irregular a partir de diversos punts.

```py
c.create_polygon(100, 300, 400, 400, 200, 0, 350,250, smoot=0)
c.create_polygon(100, 300, 400, 400, 200, 0, 350,250, fill="#ff0000", smoot=1)
```

## [`create_oval`][create_oval]

Per crear una sircumferència o el·lipsi.

```py
c.create_oval(100, 300, 400, 100, fill="blue")
```

## [`create_arc`][create_arc]

Per crear una part d'una circumferència o el·lipsi, amb diferents formats.

```py
c.create_arc(100, 000, 400, 400, start=35, extent=290, fill="yellow", style=PIESLICE)
```

## [`create_bitmap`][create_bitmap]

Per carregar una imatge de bits, predissenyada.

```py
c.create_bitmap(100, 100, bitmap="question", foreground="white", background="red")
```

## [`create_image`][create_image]

Podem utilitzar imatges ja creades en un fitxer extern, o bé, fins i tot, crear una imatge a partir d'un mapa de bits.

```py title="Suposem que tenim una imatge anomenada logo.png"
ph = PhotoImage(file="logo.png")

c.create_image(100, 300, image=ph)
```

Crearem una imatge de bits `xbm` i la carregarem per després mostrar-la.

=== "Bandera ratllada"

    ```py title
    from tkinter import *

    f = Tk()
    c = Canvas(f, width = 500, height = 500)
    c.pack()

    BITMAP = """
    --8<-- "docs/tkinter/img/imatgeSenyera.xbm"
    """

    bm = BitmapImage(data=BITMAP, foreground="red", background="white")
    c.create_image(100, 100, image=bm)
    c.create_image(164, 100, image=bm)
    c.create_image(100, 132, image=bm)
    c.create_image(164, 132, image=bm)
    c.create_image(100, 164, image=bm)
    c.create_image(164, 164, image=bm)
    ```

## [`create_text`][create_text]

Podem afegir text en el llens de la imatge. En aquest cas, ens guardarem l'objecte que creem per tal de poder veure més endavant el text que conté i fins i tot modificar-lo.

```py
t1 = c.create_text(100, 100, text = "Aquest és el text que mostrem")
## el text de l'objecte el podem recuperar
c.itemcget(t1,'text')
## modifiquem el text de l'objecte
c.itemconfigure(t1, text="Nou text")
```

## [`create_window`][create_window]

Per afegir altres finestres dins el Canvas que ja tenim. Aquesta finestra, pot contenir qualsevol tipus de giny.
