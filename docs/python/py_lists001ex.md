# Python - Exercicis de llistes

Resol aquests exercicis de llistes.

1. Escriu un programa que permeti crear una llista de paraules. El programa preguntarà un número i després demanarà tantes paraules com sigui aquest número. Finalment, el programa escriurà totes les paraules de la llista.

    Exemple d'execució:

    ```py
    Entra el número de paraules -> 6
    Entra la paraula 1 -> administració
    Entra la paraula 2 -> de
    Entra la paraula 3 -> sistemes
    Entra la paraula 4 -> informatics
    Entra la paraula 5 -> i
    Entra la paraula 6 -> xarxes
    Llista de paraules entrades
            administració
            de
            sistemes
            informatics
            i
            xarxes
    ```

2. Escriu un programa que permeti crear una llista de paraules i, a continuació, demani una paraula i digui quantes vegades tenim aquesta paraula a la llista.

    Exemple d'execució:

    ```py
    Entra el número de paraules -> 4
    Entra la paraula 1 -> dia
    Entra la paraula 2 -> setmana
    Entra la paraula 3 -> mes
    Entra la paraula 4 -> dia
    Quina paraula vols buscar? dia
    La paraula dia apareix 2 vegades a la llista.
    ```

3. Escriu un programa que permeti crear una llista de paraules i que, a continuació, demani dues paraules i substitueixi la primera ocurrència de la paraula cercada per la segona. Al final mostra com ha quedat la llista.

    Exemple d'execució:

    ```py
    Entra el número de paraules -> 7
    Entra la paraula 1 -> dia
    Entra la paraula 2 -> mes
    Entra la paraula 3 -> any
    Entra la paraula 4 -> setmana
    Entra la paraula 5 -> hora
    Entra la paraula 6 -> minut
    Entra la paraula 7 -> segon
    Quina paraula vols modificar? setmana
    Quina paraula vols modificar? trimestre
    Nova llista
    dia
    mes
    any
    trimestre
    hora
    minut
    segon
    ```

4. Escriu un programa que permeti crear una llista de paraules i que, a continuació, demani una paraula i n'elimini de la llista la primera ocurrència. Al final mostra com ha quedat la llista.

    Exemple d'execució:

    hora
    minut
    segon
    ```

    Entra la paraula 4 -> setmana
    Entra la paraula 5 -> hora
    Entra la paraula 6 -> minut
    Entra la paraula 7 -> segon
    Entra el número de paraules de la segona llista -> 3
    Entra la paraula 1 -> setmana
    Entra la paraula 2 -> trimestre
    Entra la paraula 3 -> hora
    Nova llista
    dia
    mes
    any
    minut
    segon
    ```

6. Escriu un programa que permiti crear una llista de paraules y que, a continuació, mostri les dades a l'inrevés. No s'ha de modificar la llista original.

    ```py
    Entra el número de paraules de la llista -> 4
    Entra la paraula 1 -> dia
    Entra la paraula 2 -> mes
    Entra la paraula 3 -> any
    Entra la paraula 4 -> segle
    segle
    any

    ```

7. Escriu un programa que permeti crear dues llistes de paraules i que, a continuació, escrigui les següents llistes:
    
    1. Llista de paraules que apareixen a les dues llistes.
    
    2. Llista de paraules que apareixen a la primera llista, però no a la segona.
    
    3. Llista de paraules que apareixen a la segona lista, però no a la primera.

    hora
