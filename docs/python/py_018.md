## Entrada de dades

En Informática, l'_"entrada"_ de dades, *informació*, a un programa, sovint la fem a través del teclat.

La funció [`input()`][input()] permet obtenir texte escrit pel teclat.

Quan el programa executa aquesta funció, el programa s'atura esperant que s'escrigui quelcom i es premi la tecla <kbd>intro</kbd>. Si la funció `input()` té algun texte com a paràmetre, llavors es mostrarà aquest missatge, just abans d'esperar a que l'usuari introdueixi el texte.

Vegem-ne algun exemple:

```python
a = input()     # Espera a que entrem un texte i premem intro
a               # mostrem el valor introduit anteriorment
nom = input("Introduieix el teu nom -> ")
nom             # mostrem el nom introduit anteriorment
edat = input("Introduieix la teva edat -> ")
edat            # mostrem l'edat, com a texte, introduïda anteriorment
preu = input("Introduieix un preu, amb decimals -> ")
preu            # mostrem el preu, com a texte, introduit anteriorment
```

Fixa't com el número introduït és una cadena de text, per la qual cosa no podem fer càlculs amb ell. Per convertir el text en un número cal utilitzar alguna de les funcions següents:

[`int('')`][int()] converteix una cadena de text en un nombre enter (sense decimals)

[`float('')`][float()] converteix una cadena de text en un número en coma flotant.

```python title="Per llegir nombres utilitzarem int o float juntament amb input"
edat = int(input("Introduieix la teva edat -> "))
edat            # mostrem l'edat, com a enter, introduïda anteriorment
preu = float(input("Introduieix un preu, amb decimals -> "))
preu            # mostrem el preu, com a float, introduit anteriorment
```

!!!danger "Si entrem un valor que no es pot convertir directament es produirà un ERROR"

!!!tip "Recorda que per treballar amb nombres"

    Trobarem moltes funcions per treballar amb nombres a la llibreria [`math`][math]
